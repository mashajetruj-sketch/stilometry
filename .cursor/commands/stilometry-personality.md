# /stilometry-personality
Use when explicit immutable approval has been verified and only observational language axes are requested.
Do NOT use when the request concerns diagnosis, identity, morality, protected traits, or attribution.
Run only after immutable human approval. Produce observational axes, alternatives, confidence and a not-a-diagnosis disclaimer. Reject protected traits, identity, morality, mental health and attribution.
CLI: `python stilometry.py personality runs/approved.json --output runs/personality.json`.
